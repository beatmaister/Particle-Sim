Welcome to my particle library!
    Each of the .HTML files within this folder contain a different simulation.
    All of the associated JS and CSS files are related according to "filename_script.js / filename_style.css".

Getting Started
    All that is required to run each simmulation is to open the corresponding .HTML file.
    Each File while open contain both a pause, and a close button to be used acordingly.

Prerequisites
    Please be sure all files are within the same level of the same directory

Built With
    JavaScript, HTML, CSS
